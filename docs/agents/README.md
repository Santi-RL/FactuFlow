## Cómo resolver dudas y errores
1. Buscar primero en este índice y en la documentación local usando `rg` por palabra clave.
2. Si el tema es ARCA, revisar `docs/agents/arca.md` y `docs/arca-ws/README.md`.
3. Si no hay respuesta en la documentación local, buscar en internet agregando la palabra `ARCA` al error o tema.
4. Si con `ARCA` no aparece, repetir la búsqueda usando `AFIP`.
